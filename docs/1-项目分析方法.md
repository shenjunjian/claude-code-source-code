## 📚 Claude Code 源码阅读指南

### 1. 项目概述

这是 **Claude Code** 的源码 —— 一个基于 TypeScript + React (Ink) 构建的 AI 编程助手 CLI 工具。

**技术栈**:
- **运行时**: Node.js / Bun
- **UI框架**: React + Ink (终端UI)
- **CLI框架**: Commander.js
- **语言**: TypeScript

---

### 2. 核心目录结构

```
src/
├── main.tsx              # 🚀 主入口文件
├── commands/             # 📋 CLI 子命令实现
├── components/           # 🎨 React UI 组件
├── services/             # 🔧 核心服务
├── utils/                # 🛠️ 工具函数
├── hooks/                # ⚛️ React Hooks
├── ink/                  # 📺 终端UI框架
├── tools/                # 🔨 AI 工具定义
├── state/                # 📦 状态管理
├── constants/            # 📝 常量定义
├── context/              # 🔄 React Context
├── entrypoints/          # 🚪 不同入口点
├── migrations/           # 📦 数据迁移
└── bootstrap/            # 🔄 启动引导
```

### 3. 入口文件 `main.tsx` 执行流程

```
┌─────────────────────────────────────────────────────────────┐
│  1. 启动性能分析 & 预加载                        │
│     - profileCheckpoint() - 性能标记                         │
│     - startMdmRawRead() - MDM 配置预读                       │
│     - startKeychainPrefetch() - 钥匙串预取                   │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 大量模块导入 (~135ms)                                    │
│     - Commander.js, React, 工具函数, 服务模块等              │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  3. main() 函数                                             │
│     - 安全设置                         │
│     - 信号处理                                   │
│     - 深度链接处理                               │
│     - SSH/Direct Connect 模式检测                           │
│     - 交互模式判断                            │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  4. run() 函数 - Commander 命令解析                          │
│     - 创建 CLI 程序实例                                      │
│     - 注册全局选项 (--debug, --print, --model 等)           │
│     - preAction 钩子: init(), 迁移, 设置加载                │
│     - 注册子命令                                             │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  5. action 处理器                                           │
│     - setup() - 完整初始化                                   │
│     - 启动 REPL (交互模式) 或 runHeadless() (打印模式)       │
└─────────────────────────────────────────────────────────────┘
```

### 5. 关键概念

#### 📋 命令系统
```typescript
// commands.ts 中注册子命令
program.command('mcp').description('MCP server management')
program.command('config').description('Manage settings')
// ... 更多命令
```

#### 🔧 工具系统
AI 可以调用的工具，如 `Bash`, `Edit`, `Read`, `Write` 等，定义在 `tools/` 目录。

#### 📡 MCP (Model Context Protocol)
允许扩展 AI 能力的协议，实现在 `services/mcp/`。

#### 🔄 状态管理
使用 React Context + 自定义 store，入口在 `state/` 目录。
